HTML Table Reference

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<tr> Table Row Used to add rows to a table before adding table data and table headings.
<td> Table Data Nested inside a table row element to add a cell of data to a table.
<thead> Table Head Defines the headings of table columns encapsulated in table rows.
<tbody> Table Body A semantic element that will contain all table data other than table heading and table footer content
<tfoot> Table Foot Uses table rows to give footer content or to summarize content at the end of a table.

Table Attributes

Attribute Name Description
colspan Column Span The colspan attribute on a thead or td element indicates how many columns that particular cell should span within the table. Colspan default value is 1 by default and will take any positive integer between 1 and 1000.
rowspan Row Span The rowspan attribute on a table header or table data element indicates how many rows that particular cell should span within the table. The rowspan default value is 1 and will take any positive integer up to 65534.
